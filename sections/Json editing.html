How to Edit Json Files

Learn how to edit json files using these json commands for mc3ds, each command will contain a description and snippet of code

Name Function Code
Is_baby This command is used for component groups which reflect baby varients. "minecraft:is_baby": { }
Scale This command is used to rescale a compontent groups model, such as babies. "minecraft:scale": {"value": X}
Movement This command is used to set a compontent groups movement speed. "minecraft:movement": { "value": X}
Equipment This command is used to set a compontent groups held weapon or armor, such as skeletons. "minecraft:equipment": { "table": "loot_tables/entities/X.json"}
Nameable This command controls whether the mob can be named using a nametag. "minecraft:nameable": { }
Health This command controls the max and default health of a mob. "minecraft:health": {"value": X,"max": Y}
Collision_box This command defines the collision box of the mob.(usually around 1 block) "minecraft:collision_box": {"width": X,"height": Y}
Identifier This command is neccessary to label the mob to be referenced in other commands/jsons. "minecraft:identifier": {"id": "minecraft:X"}
Type_family This command specifies what familes the mob is a part of.(can be multiple) "minecraft:type_family": {"family": [ "X",Y,Z,etc...]}
